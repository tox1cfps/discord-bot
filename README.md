# 🤖 Discord Bot (Projeto Cômico em Python)

Este é um bot para Discord desenvolvido em **Python**, criado com o objetivo de ser um projeto divertido, experimental e com funções cômicas para uso em servidores do Discord.

O projeto foi feito como forma de estudo, prática com APIs, automação e integração com inteligência artificial, além de servir como um bot interativo para amigos e comunidades.

---

## 🚀 Funcionalidades

### 🔍 `!pesquisar`
Comando que utiliza um **agente de Inteligência Artificial** para responder perguntas e pesquisar informações automaticamente.

**Exemplo:**

!pesquisar Quem é o neymar?

📰 !jornal

Lê as mensagens recentes do chat e cria um jornal fictício e humorístico, como se fosse uma notícia baseada nos acontecimentos do servidor.

📦 Como Executar
1️⃣ Clone o repositório
git clone https://github.com/tox1cfps/discord-bot.git
cd discord-bot
2️⃣ Instale as dependências
pip install -r requirements.txt
3️⃣ Configure o token do bot

Crie um arquivo .env ou configure diretamente no código:

DISCORD_TOKEN=seu_token_aqui
4️⃣ Execute o bot
python main.py
